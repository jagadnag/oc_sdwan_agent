# SD-WAN Troubleshooting

## Systematic Approach
1. Gather information
2. Identify scope
3. Analyze symptoms
4. Test hypothesis
5. Implement solution
6. Verify fix
7. Document findings

## Data Collection
- Device logs and diagnostics
- Control plane status
- Policy configuration
- Transport status
- Application metrics

## Common Tools
- vManage dashboards
- Device CLI
- Packet captures
- Log analysis
