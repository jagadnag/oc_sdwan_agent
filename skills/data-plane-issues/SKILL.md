# Data Plane Issues

## Connectivity Failures
- No tunnels established
- Selective tunnel failures
- Packet loss
- High latency
- Tunnel flapping

## Diagnostics
- Check tunnel status
- Monitor BFD sessions
- Verify encryption
- Check MTU
- Monitor interface errors

## Resolution
- Establish tunnels
- Fix underlying transport
- Adjust parameters
- Validate routing
- Monitor recovery
